GIT

Git ₍ₒᵤ em inglês britânico₎ é um sistema de controle de versões distribuído, usado principalmente no desenvolvimento de software, mas pode ser usado para registrar o histórico de edições de qualquer tipo de arquivo

  • sistema de controlo de versoes
  • gratuito
  • open source
  • rapido
  • sistema distribuido, ou seja, cada utilizador pode ter uma copia do repositorio localmente
  • é o mais utilizado

comando para ver se temos instalado e qual a versao: git --version

para iniciar um repositorio local: git init

para consultar o status dos ficheiros: git status

para adicionar ficheiro ao staging area / area de preparaçao: git add intro_git.html

para configurar user name: git config --global user.name "Joana Meneses"

para configurar user email: git config --global user.email joana.p.s.m@gmail.com

para fazer commit: git commit -m "commit inicial"

para ver historico de commits: git log

git add ficheiro1 ficheiro2 para adicionar ficheiros

git add . para adicionar todos os novos ficheiros do diretorio

para guardar versao do programa ou efetuar snapshot: git commit -m "pequena descriçao"


git log --oneline --graph para ver o historico de forma diferente

ramificaçao do codigo

listar os ramos git branch

criar ramo git branch novodesign

trocar de ramo git switch

eliminar ramo: git branch -D novodesign

fazer merge entre ramos git merge ramoajuntar
